# AWS Glue Crawler

---

## AWS Glue Crawler 란?

AWS Glue Crawler 는 AWS Glue 에서 사용할 데이터를 자동으로 탐색하고, 데이터의 스키마를 추출하는 기능을 제공하는 서비스이다.

## AWS Glue Crawler 사용법

### AWS Glue Crawler 생성

AWS Glue Crawler 를 생성하기 위해 AWS Glue Console 에서 Data Catalog > Crawlers 로 들어간다.
